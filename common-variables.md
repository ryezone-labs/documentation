Common Variables
================

## Networking

- `ryezone_labs_lanInterface` (string)

   Sets the LAN interface on the edge node.

   Usages:
     - [ryezone_labs.dhcp](https://github.com/ryezone-labs/dhcp)

- `ryezone_labs_wanInterface` (string)

   Sets the WAN interface on the edge node.

   Usages:
     - [ryezone_labs.dhcp](https://github.com/ryezone-labs/dhcp)
